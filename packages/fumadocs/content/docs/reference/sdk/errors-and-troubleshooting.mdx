## Common CLI Errors

### Missing base URL

Error:

`Missing base URL. Pass --base-url explicitly or set MONOLAYER_BASE_URL.`

Fix:

- Pass `--base-url https://control-plane-domain`, or
- Set `MONOLAYER_BASE_URL`.

### Missing auth token (`projects:list`)

Error:

`Missing auth token. Pass authToken explicitly or set MONOLAYER_AUTH_TOKEN`

Fix:

- Pass `--auth-token`, or
- Set `MONOLAYER_AUTH_TOKEN`.

### Invalid deployment token (`deployments:deploy`)

Error:

`auth-token must start with "deploy_token_"`

Fix:

- Use a deployment token prefixed with `deploy_token_`.

### Branch resolution failure

Error:

`Unable to determine current git branch...`

Fix:

- Run inside a git repository with a checked-out branch, or
- Pass `--branch-name`.

## Localhost HTTPS Fetch Failure

If the base URL is `https://localhost:<port>` and fetch fails:

- If your local server is HTTP-only, use `http://localhost:<port>`.
- If HTTPS is enabled with self-signed certs, trust the certificate in your local environment.

## Debug Mode

For deployment command request debugging:

```bash
DEBUG=1 npx monolayer deployments:deploy ...
```

This prints request details with redacted auth token.
